# Synapta CLI

## Installation (Linux/macOS)

```bash
curl -fsSL https://raw.githubusercontent.com/ashiqrniloy/synapta-cli/main/scripts/install.sh | sh
```

## Update

Use the same command to update to the latest release:

```bash
curl -fsSL https://raw.githubusercontent.com/ashiqrniloy/synapta-cli/main/scripts/install.sh | sh
```
